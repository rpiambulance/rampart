# Rampart

A NodeJs backend for RPI Ambulance.



## Getting started

### Step 1 : Make sure you have docker and docker-compose installed
[Docker](www.docker.com)

### Step 2 : Copy the env file and set it up

```bash
$ cp .env.example .env
```

### Step 3 : Start the backend

```
$ ./scripts/init.sh
```
